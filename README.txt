AdmonCundech - Sitio web listo para GitHub Pages

Contenido:

- index.html
- /img/ (contiene imágenes placeholder)

Instrucciones rápidas:

1) Reemplazar Formspree:
   - Si quieres recibir mensajes del formulario, crea una cuenta en https://formspree.io/ y reemplaza la acción del formulario en `index.html` con tu form ID.

2) Descargar imágenes reales desde Google Drive (opcional):
   - Ejecuta el script `download_images.sh` (requiere curl).
   - Esto descargará las imágenes desde tus enlaces de Google Drive y las guardará en la carpeta `img/`.

3) Publicar en GitHub Pages:
   - Crea un repositorio en GitHub y sube todos los archivos de esta carpeta.
   - En Settings > Pages, activa deployment desde la rama `main` o `master` y selecciona la carpeta `/ (root)`.

Archivos incluidos:
- index.html
- img/ (7 placeholders)
- download_images.sh (script para obtener las imágenes desde Drive)
- LICENSE and README

Si quieres que yo genere también la página en GitHub (crear repo y publicar), necesitarás darme acceso a tu cuenta (no es posible). Puedo guiarte paso a paso para publicarlo.
